# Virtual Column in pandas DataFrame

This repository contains my solution for the Data Engineering task. 

The `add_virtual_column` function safely evaluates mathematical expressions and adds the result as a new virtual column to an existing pandas DataFrame. It includes built-in data validation steps to check column labels and mathematical operators before executing any calculations.

## Project Structure
* `solution.py` - Contains the main function and validation logic.
* `test_virtual_column.py` - Unit tests provided to verify the function's behavior.
